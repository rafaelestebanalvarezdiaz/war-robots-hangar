# War Robots – Hangar Personal

Este proyecto es una página web básica desarrollada con HTML donde se presenta una selección personal de robots del videojuego War Robots, incluyendo un titán favorito y los robots principales del hangar.

## Tecnologías utilizadas
- HTML5
- Git
- GitHub
